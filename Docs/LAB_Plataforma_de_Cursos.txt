Modelo  de  Dados  
1  Tabela  Usuarios  
●  ID_Usuario (PK  -  Chave  Primária)  ●  NomeCompleto  ●  Email (Unique)  ●  SenhaHash  ●  DataCadastro  
2.  Tabela  Categorias  
●  ID_Categoria (PK)  ●  Nome (Unique)  ●  Descricao  
3.  Tabela  Cursos  
●  ID_Curso (PK)  ●  Titulo  ●  Descricao  ●  ID_Instrutor (FK  -  Chave  Estrangeira  para  Usuarios)  ●  ID_Categoria (FK  para  Categorias)  ●  Nivel (Ex:  Iniciante,  Intermediário,  Avançado)  ●  DataPublicacao  ●  TotalAulas  ●  TotalHoras  
4.  Tabela  Modulos  
●  ID_Modulo (PK)  ●  ID_Curso (FK  para  Cursos)  ●  Titulo  ●  Ordem (Para  sequenciar  os  módulos  dentro  do  curso)  
5.  Tabela  Aulas  
●  ID_Aula (PK)  ●  ID_Modulo (FK  para  Modulos)  ●  Titulo  ●  TipoConteudo (Ex:  Vídeo,  Texto,  Quiz)  ●  URL_Conteudo  ●  DuracaoMinutos  ●  Ordem (Para  sequenciar  as  aulas  dentro  do  módulo)  
6.  Tabela  Matriculas  
●  ID_Matricula (PK)  ●  ID_Usuario (FK  para  Usuarios)  ●  ID_Curso (FK  para  Cursos)  ●  DataMatricula  ●  DataConclusao (Nulável)  
7.  Tabela  Progresso_Aulas  
●  ID_Usuario (PK,  FK  para  Usuarios)  ●  ID_Aula (PK,  FK  para  Aulas)  ●  DataConclusao  ●  Status (Ex:  Concluído)  
8.  Tabela  Avaliacoes  
●  ID_Avaliacao (PK)  ●  ID_Usuario (FK  para  Usuarios)  ●  ID_Curso (FK  para  Cursos)  ●  Nota (Ex:  1  a  5)  ●  Comentario (Nulável)  ●  DataAvaliacao  
9.  Tabela  Trilhas  
●  ID_Trilha (PK)  ●  Titulo  ●  Descricao  ●  ID_Categoria (FK  para  Categorias)   
10.  Tabela  Trilhas_Cursos (Tabela  Associativa)  
●  ID_Trilha (PK,  FK  para  Trilhas)  ●  ID_Curso (PK,  FK  para  Cursos)  ●  Ordem (Para  definir  a  sequência  dos  cursos  na  trilha)  
11.  Tabela  Certificados  
●  ID_Certificado (PK)  ●  ID_Usuario (FK  para  Usuarios)  ●  ID_Curso (FK  para  Cursos)  curso.  ●  ID_Trilha (FK  para  Trilhas,  Null)  ●  CodigoVerificacao (Unique)  ●  DataEmissao  
12.  Tabela  Planos  
●  ID_Plano  (PK)  ●  Nome (not  Null)  ●  Descricao  ●  Preco (not  Null)  ●  DuracaoMeses (not  Null)  
13.  Tabela  Assinaturas  
●  ID_Assinatura  (PK)  ●  ID_Usuario (FK  para  Usuarios,  not  Null )  ●  ID_Plano (FK  para  Planos,  not  Null ) ●  DataInicio  (not  Null)  ●  DataFim (not  Null)  ●  DataFim (not  Null)  
13.  Tabela  Pagamentos  
●  ID_Pagamento  (PK)  ●  ID_Assinatura (FK  para  Assinaturas,  not  Null )  ●  ValorPago (FK  para  Planos,  not  Null ) ●  DataPagamento  (not  Null)  ●  MetodoPagamento  (not  Null)  ●  Id_Transacao_Gateway  (not  Null)  ●  DataFim (not  Null)
